#Introdução

Este relatório apresenta o processo de implementação de um código em python d desafio de projeto para utilizando os conceitos ensinados do curso para uma pequena automação via software utilizando comando por voz para abertura de uma aplicação no computador, para exemplo foi  utilizado  o Word.

#Metodologia do Trabalho

O código em python foi realizado no Vscode mas pode ser executado em outros softwares python disponíveis no mercado.

#Depencias da aplicação

Deverá ser instalado os seguintes complementos no python.
speech_recognition
subprocess
pyaudio

#Uso da Aplicação

Esteja em um local que não tenha muito barulho ao executar o teste da aplicação. 
Ao executar o código deverá ser usado a frase “Abrir Word” , se a fala foi compreendida corretamente a software Word vai abrir.

Obs. Eventualmente o antivírus pode solicitar permissão para rodar o pyaudio.
